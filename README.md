# 🗺️ Sistema de Gestão Territorial - Boa Vista

Este repositório é uma plataforma técnica para a **Rede de Proteção Social**, consolidando mapas interativos e formulários digitais.

## 🏘️ Mapa Vila Jardim
O sistema inicia com o **Mapa Vila Jardim**, permitindo a navegação detalhada pelas unidades habitacionais do conjunto. É a ferramenta principal para o planeamento de visitas e acompanhamento territorial.

## 📍 Redes Disponíveis
- **CRAS/CREAS**: Proteção social básica e especial.
- **Conselho Tutelar**: Áreas de abrangência e garantia de direitos.
- **RAPS**: Rede de saúde mental.
- **Defensoria**: Assistência jurídica gratuita.

## 📄 Ficha Digital
A **Ficha de Atendimento (CAS)** está integrada no botão azul. Pode ser preenchida e impressa diretamente do navegador (Ctrl+P), eliminando a necessidade de software externo.

---
**Dica:** Use o campo de busca no canto superior esquerdo de cada mapa para localizar ruas ou equipamentos específicos.
